# The New School IRL

## Feature: Browse Classes

### Non-Technical Summary
This feature is like a smart search engine for your school catalog, letting you quickly find and filter exactly the right classes to take next semester.

### What it Does
The "Browse Classes" feature lets students effortlessly search, filter, and discover available courses through an intuitive and dynamic interface. It provides both a quick search bar for knowing precisely what you want, and a robust advanced search form for exploring options by various criteria like college, campus, modality, and level.

### How to Use It
1. **Navigate to the Catalog**: Tap the "BROWSE CLASSES" tab in the Quick Links horizontal navigation bar at the top of the screen.
2. **Basic Search**: Use the main search bar to look up courses by keyword, CRN, or department name.
3. **Advanced Filtering**: Click on the red "Advanced Search" toggle link below the search bar to expand the detailed filter menu.
4. **Apply Filters**: Select your desired College, Term, Campus, Modality, Level, and Status using the dropdowns and radio buttons, then tap the red "FILTER" button to view tailored results.
5. **View Results & Add**: Browse the resulting course cards. Each card cleanly displays the course code, term, title, meeting times, and instructor. Tap the `+` button on any course card's right edge to interact with that course.
